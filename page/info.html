간편하게 데이터를 직접 분석할 수 있는 서버리스 대화형 쿼리 서비스

대규모 정형/비정형 데이터 분석을 표준 SQL사용을 통해 쉽고 효율적으로 실행합니다.

  • 빠르고 간편한 비즈니스 인사이트 생성

    SQL 사용을 통해 누구나 쉽고 빠르게 대규모 데이터 세트를 분석할 수 있으며, 고객의 다양한 니즈에 맞는 비즈니스 인사이트를 제공합니다.

  • 서버리스 데이터 분석 서비스

    분석 작업에 필요한 별도의 인프라 리소스 관리가 요구되지 않으며, 데이터 분석에만 집중할 수 있습니다. 또한, 실행된 쿼리만큼 요금이 청구되어 비용 효율적인 사용이 가능합니다

  • NAVER Cloud Data 서비스 통합 분석

    NAVER Cloud의 Data Catalog와 애널리틱스 서비스를 이용하여 Storage에 저장되어 있는 데이터를 간편하게 조회 및 분석할 수 있습니다.

상세기능

  • 대규모 데이터세트 분석을 위한 편리한 쿼리 환경

    대규모 데이터 분석에 최적화된 SQL엔진인 Trino를 사용하며, 사용자는 NAVER Cloud 콘솔에서 제공하는 SQL Editor를 사용하여 쉽고 빠르게 SQL 쿼리를 작성하고 실행 결과를 확인할 수 있습니다.

  • Data Catalog를 활용한 통합 데이터 분석

    다양한 데이터가 통합된 메타데이터 레포지토리를 Data Catalog를 사용해 원본 데이터에 접근합니다. 또한, Data Flow의 완전관리형 ETL 기능을 사용하면 데이터를 변환하거나 데이터를 컬럼 형식으로 변환하여 쿼리 성능 최적화를 통한 비용 절감 효과를 가져오게 됩니다.

  • 효과적인 SQL 쿼리 관리

    효과적인 SQL 쿼리 관리는 대규모 데이터를 분석하는데 있어 중요합니다. Data Query가 제공하는 Queries는 사용자가 작성한 쿼리를 재사용 가능하도록 관리 및 공유 기능을 지원합니다. 또한 Query History를 통해 실행한 쿼리 상세 정보를 확인할 수 있습니다.

  • 서버리스 제공을 통한 유연한 확장 및 높은 가용성

    서버리스 환경으로 제공되는 Data Query는 자동으로 확장되며, 데이터 세트와 쿼리 복잡도에 최적화된 쿼리를 실행합니다. 또한, 쿼리 실행 시 리소스 인프라 상황에 맞는 자동 라우팅을 통해 높은 가용성을 보장합니다.

요금 예시

  • 예시 1) Object Storage에 저장된 30.128GB의 파일 데이터를 총 6회 가져오도록 쿼리를 실행 했을 경우,
    30.128GB X 6회 = 180.768GB → 180.77GB, 180.77GB X 5.86원 = 1,050원
  • 예시 2) 3일동안 첫째 날 100GB, 둘째 날 200GB, 셋째 날 300GB의 파일 데이터를 가져오도록 쿼리를 실행 했을 경우
    100GB X 5.86원 + 200GB X 5.86원 + 300GB X 5.86원 = 3,510원
