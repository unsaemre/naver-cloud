HyperCLOVA X

기술, 데이터 그리고 노하우의 집약체

HyperCLOVA X는 AI 백본 모델, 데이터, 튜닝 기술, 슈퍼 컴퓨팅 인프라와 이 모든 것에 대한 노하우로 만들어집니다.HyperCLOVA의 업그레이드 모델인 HyperCLOVA X는 LLM의 한계인 사실 왜곡(hallucination), 시의성 정보 취약, 수식 연산&계산 취약, 정의하지 않은 정보 제공, 이미지 생성&해석의 어려움을 보완하여 더욱 놀라운 성능과 향상된 품질을 제공합니다.

* LLM (Large Language Model): 대용량의 텍스트 데이터를 학습시켜 성능을 극대화한 언어 모델.주어진 문장에서 다음 단어를 맞출 수 있도록 학습시킴

비즈니스에 최적화된 초대규모 AI 개발 도구

고객이 HyperCLOVA X 언어 모델을 이용한 AI 서비스를 만들 수 있게 해 주는 개발 도구입니다.CLOVA Studio를 통해 기업이 자체 보유한 데이터를 결합하고 이용자의 상황에 최적화된 AI 서비스를 구현할 수 있습니다.SFT, RLHF 등 업그레이드된 튜닝 기술과 튜닝 API, 임베딩 API와 같은 유연한 개발 도구를 제공하며, 기업 내/외부 서비스 API와 HyperCLOVA X 연동을 단순화하는 Skill Trainer를 통해 다양한 전문 정보를 손쉽게 활용할 수 있습니다.

* SFT (Supervised fine-tuning): 사용자의 의도를 더 잘 이해시키기 위해 고품질의 데이터셋으로 파인튜닝하는 방식* RLHF (Reinforcement Learning from Human Feedback): 사람의 피드백을 통해 AI 모델의 성능을 강화하는 방식

전용 클라우드 인프라를 제공하는
CLOVA Studio Exclusive

사용자의 활용 의도에 맞게 특화 모델을 만들 수 있는 SFT 튜닝과 전용 인프라를 함께 제공하는 서비스입니다. 고객별로 할당된 전용 인프라는 실시간 응답 속도를 안정화하고 사용량을 보장합니다. 자체 인프라 구축 없이도 강력한 SFT 모델을 활용한 실시간 AI 서비스를 개발할 수 있습니다.

HyperCLOVA X에 최적화된 하이브리드 클라우드

데이터센터 각의 기술을 집약시킨 하이브리드 클라우드 전용 하드웨어 Neurocloud 위에 GPU 클러스터를 결합하여, 초대규모 AI 모델 HyperCLOVA X 와 학습, 운영 도구를 함께 제공하는 상품입니다.고객의 데이터를 학습한 특화 언어 모델을 만들고, 고객의 데이터센터 내에서 사내망과 연동된 네트워크 환경을 구성하여 보안 요건을 완벽하게 충족시킬 수 있습니다.
